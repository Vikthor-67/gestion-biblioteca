# Gestión de Biblioteca

Aplicación de gestión de biblioteca que permite administrar la información básica del sistema.

## ¿Qué puede hacer la app actualmente?
- Registrar libros.
- Listar/consultar libros.
- Buscar libros por datos del registro (por ejemplo: título, autor o código/ISBN).
- Editar la información de un libro.
- Eliminar libros.

- Registrar usuarios/socios.
- Listar/consultar usuarios.
- Editar usuarios.
- Eliminar usuarios.

- Registrar préstamos.
- Registrar devoluciones.
- Consultar préstamos (activos o historial, según lo que tengas implementado).

## Notas
Este repositorio describe únicamente las funcionalidades implementadas actualmente en el código.
